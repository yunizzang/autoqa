# AutoQA 🧠
AI 기반 테스트 케이스 자동 생성 및 실행 시스템

## 📌 프로젝트 개요
AutoQA는 QA 테스트 자동화를 위해 개발된 Python 기반 플랫폼으로,
테스트 케이스 생성부터 실행, 리포트까지 자동화합니다.

## ⚙️ 기술 스택
Python | pytest | Selenium | Requests | Allure | GitHub Actions | OpenAI API

## 🚀 주요 기능
- AI 기반 테스트케이스 생성
- API 및 UI 자동화 테스트
- CI/CD 파이프라인 자동 실행
- Allure Report 기반 시각화 리포트

## 📈 기대 효과
- 테스트 작성 시간 60% 단축
- 테스트 커버리지 향상
- QA 자동화 파이프라인 정립

## 📂 프로젝트 구조
(위 폴더 트리 삽입)

## 🧩 실행 방법
pip install -r requirements.txt
pytest --alluredir=reports/allure-results
allure serve reports/allure-results
